Removed the web frameworks[edit]

I've been so bold as the remove the "frameworks" (i.e. web frameworks) from this template. They seem to suggest that Python is primarily a web programming language, like PHP. Python, however, was conceived as a scripting language and still very often used as one. It is also a major scientific programming language, so listing web frameworks but not NumPy, SciPy, SAGE and SymPy is quite unfair. Qwertyus (talk) 22:36, 4 September 2012 (UTC)
